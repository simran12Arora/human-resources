# human-resources sql  analysis

There are three tables in this dataset: "departments," "employees," and "projects." In this context, various SQL functions are employed, including:

Joins: These are used to combine data from multiple tables, allowing for comprehensive analysis and reporting.

Order by Clause: It is utilized to sort query results based on specific columns, facilitating the arrangement of data in ascending or descending order.

Subquery: Subqueries are employed to retrieve data from one table based on values from another table, enabling complex data extraction and filtering.

Window Function: Window functions are applied to a set of rows related to the current row, providing aggregated or calculated results over specific partitions or orderings, enhancing data analysis capabilities.

The database server used for this dataset is PostgreSQL, a powerful and open-source relational database management system.
